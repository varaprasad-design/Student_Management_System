# Student Management System

A CRUD-based application developed using C#, .NET Core, and SQL Server to manage student records efficiently.

## Features
- Add, update, delete, and search student records
- Structured and modular code using OOP principles
- Secure and optimized database interactions

## Technologies Used
- C#
- .NET Core
- ADO.NET
- SQL Server

## Key Learnings
- Hands-on experience with backend development
- Database connectivity using ADO.NET
- Writing optimized SQL queries
- Applying object-oriented programming concepts

## Future Enhancements
- User authentication and role-based access
- Web-based UI using ASP.NET MVC
